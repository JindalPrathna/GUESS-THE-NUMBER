# GUESS THE NUMBER 
A Simple Python game where the player tries to guess a randomly generated number.
## How to Run
1.   Install Python.
2.   Download the Project.
3.   Open a terminal in the Project folder.
4.   Run:
   Python miniproject.py
## Features
- Randon number generation
- User Input
- Guess checking
- Simple game Logic
